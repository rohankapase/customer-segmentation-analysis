## Project Overview
This project focuses on performing **Customer Segmentation Analysis** for an e-commerce business using data analytics and machine learning techniques.  
The main goal is to group customers into meaningful segments based on their **purchase behavior and spending patterns**, which helps businesses create targeted marketing strategies and improve customer satisfaction.

---

## Objectives
- Analyze customer purchase behavior
- Segment customers using clustering techniques
- Visualize customer segments
- Provide business insights and recommendations

---

## Dataset
The dataset contains customer-related information such as:
- Annual Spending
- Purchase Frequency
- Total Orders

> The dataset can be obtained from Kaggle or generated manually for learning purposes.
